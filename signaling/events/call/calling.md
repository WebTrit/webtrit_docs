# `calling` call event

An [`outgoing_call`](../../requests/call/outgoing_call.md) request starts processing notification.

## Additional event properties

None.

```json
{
  "event": "calling",
  "transaction": "transaction-1",
  "line": 0,
  "call_id": "qwertyuiopasdfghjklzxcvbnm"
}
```
